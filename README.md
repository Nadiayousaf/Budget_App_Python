# Python Budget Tracker App 💵

A simple, easy-to-use Python app to manage your personal finances. Track your income, log your expenses, and see your total balance in real-time.

## Features ✨
* **Add Income:** Keep a record of your earnings.
* **Add Expenses:** Log what you spend with simple categories.
* **Check Balance:** Instantly calculate how much money you have left.
* **Data Storage:** Save your money logs to keep a record.

## How to Use 🚀
1. Open your terminal or command prompt.
2. Clone this repository:
   `git clone https://github.com`
3. Run the Python script:
   `python main.py`
4. Follow the on-screen menu to manage your money!
